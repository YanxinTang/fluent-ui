## Introduction

A Fluent UI toolkit implements by Vue 2.

## Installation

```sh
yarn add @tyx1703/fluent-ui
npm i @tyx1703/fluent-ui
```

## Usage

### Fully import

```js
import FluentUI from '@/tyx1703/fluent-ui';

Vue.use(FluentUI);
```
